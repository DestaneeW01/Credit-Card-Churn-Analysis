# Credit Card Churn Risk Analysis

## Project Overview
This project analyzes credit card customer behavior to identify factors associated with customer churn and segment customers by churn risk.

## Tools Used
Python – Data cleaning and exploratory analysis  
SQL – Data querying and churn segmentation  
Tableau – Data visualization and dashboard development

## Key Findings
- Overall churn rate: 16.07%
- High-risk customers show a 37% churn rate
- Lower transaction activity strongly correlates with customer attrition
- High-risk customers also have more frequent customer service contacts

## Project Workflow
1. Data cleaning and exploratory analysis in Python
2. SQL queries to calculate churn metrics and risk segments
3. Tableau dashboard to visualize churn risk patterns
